This web page is a simple demo of how you can use a NoSQL database like neo4j to build a book collaborative filtering recommendation system. The data set used can be found here. In the "search by user" section you can enter a user's code and obtain a list of suggested books based on similarities with other users. The similarity score is a score that was obtained through algorithms such as FastRP and KNN, for more information see here. In the other section you can simply search book by title. In "Top 10" you can get the most rated book for the first 10 countries in the dataset {% endblock %}
